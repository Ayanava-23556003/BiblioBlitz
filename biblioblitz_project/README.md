## 🚀 Quick Start

No prerequisite installations or runtime environments are required! Simply capture the compilation setup package directly from the repository allocation space to get started:

* **Step 1:** Download the executable installation wizard directly from the official repository path: **[BiblioBlitz_Setup_v4.1.exe](https://github.com/Ayanava-23556003/BiblioBlitz/releases/download/v4.1.0/BiblioBlitz_Setup_v4.1.exe)**
* **Step 2:** Run the setup installer file to automatically configure your application icon links, shortcut trajectories, and workspace parameters.
* **Step 3:** Open the suite interface, establish your verification parameters, target your chosen repositories, and begin compiling your datasets.

## What's New in v4.1

| Feature Module | Operational Enhancements |
| :--- | :--- |
| **Light-Beige Theme UI** | Transitioned to a warm, eye-friendly light palette theme optimizing visual contrast and readability for extended review sessions. |
| **Multi-Select Demographics** | Upgraded geographic query controls from singular entries to an expandable multi-selection array covering international boundaries. |
| **Publisher-Grouped Journals** | Live journal search queries automatically aggregate and display matching publication venues organized under their respective Publication Houses. |
| **Granular Hydro-Spatial Fields** | Embedded input fields for State/Province boundaries and Hydrological Basin/Watershed zones to narrow search parameters. |
| **Contextual Upload Operations** | Outfitted data acquisition and analysis panels with dedicated local file/folder upload buttons to streamline file integration. |
| **Unified 3-Tab Architecture** | Consolidated search workflows into a modular tabbed interface: Tab 1 for PDF/CSV Data Acquisition, Tab 2 for Systematic Literature Review (SLR), and Tab 3 for interactive Matplotlib data visualizations. |
